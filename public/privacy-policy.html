Privacy Policy

Effective date: June 18, 2026

This Privacy Policy describes how the Custom Automations Discord bot ("Custom Automations", "we", "our", or "us") collects, uses, stores, and protects information when used in Discord servers.

Information We Collect

We collect only data needed to operate moderation and automation features, including:

  • Discord IDs (user IDs, guild IDs, role IDs, channel IDs, message IDs where required for moderation records)
  • Moderation metadata (action type, moderator ID, target ID, reason, timestamps)
  • Configuration data set by server staff (allowed roles, log channels, boost channel settings)
  • Bloxlink integration data required for moderation workflows (linked Roblox ID lookups and related cache/history)

We do not intentionally collect personal data beyond Discord-provided identifiers and information necessary to provide bot functionality.

How We Use Information

We use collected data solely to provide and maintain bot features, including:

  • Executing moderation commands (ban, unban, timeout, untimeout, purge, logging)
  • Applying role-based command access and admin controls
  • Maintaining moderation history for transparency and accountability
  • Detecting and responding to ban-evasion patterns through linked-account checks
  • Delivering server automations configured by authorized server staff

Message Content

If Message Content intent is enabled, message content is processed only for operational bot features such as prefix command handling and configured moderation automations. We do not use message content for advertising, profiling, or unrelated analytics.

Data Sharing

We do not sell user data. We share data only with service providers or APIs necessary to run requested features (for example, Bloxlink API for linked-account lookups), and only to the extent required for those features.

Data Retention

Data is retained only as long as needed for moderation, operational, and audit purposes, or until removed by bot maintainers as part of maintenance and cleanup.

Security

We use reasonable technical and organizational safeguards to protect stored data. No method of storage or transmission is guaranteed to be 100% secure.

Your Choices

Server administrators control whether to add or remove the bot from their servers. If a server removes the bot, future processing for that server stops. Requests regarding server data can be made through the contact method below.

Children's Privacy

The bot is intended for use in Discord environments and is not directed to children under 13. If you believe data from a child has been collected in error, contact us to request deletion.

Changes To This Policy

We may update this Privacy Policy from time to time. The latest version will be posted at this page with an updated effective date.
