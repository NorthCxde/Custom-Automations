Commands

  • ?read — list channels
  • ?ping — check latency
  • ?reload — reload commands
  • ?ban / ?unban — moderation
  • ?mute / ?unmute — timeout moderation
  • /purge count:<number> [user:@user] [user2:@user] [user3:@user] [user4:@user] — delete recent messages from the channel or from multiple users
  • /ban user:@user [user2:@user] [user3:@user] [user4:@user] [reason:text] — ban one or more users from the server
  • /mute user:@user [user2:@user] [user3:@user] [user4:@user] duration:<time> [reason:text] — timeout one or more users for a duration
